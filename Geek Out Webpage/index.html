Geek-Out is a transformation of the card-based party game, Geek Out!, an application that can be played virtually. Players are able to join virtual game rooms via a room code and play width friends no matter where they are!

Geek-Out implements a simple user interface that allows players to play their favorite card game without feeling confused or indifferent as the application is extremely easy to use.
